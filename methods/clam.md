 1407: H    MetaWorld Dataset Generation
 1408: TD-MPC [23] is a model-based reinforcement learning (RL) algorithm that performs local trajectory
 1409: optimization in the latent space of a learned world model. TD-MPC2 [37] provides a series of
 1410: improvements over TD-MPC2 including architectural modifications and other design choices. All
 1411: components of TD-MPC2 are implemented as MLPs with linear layers followed by LayerNorm and
 1412: Mish activations. TD-MPC2 uses an ensemble of 5 Q-functions to to reduce bias in the TD-targets.
 1413: It learns a closed-loop control policy by planning using a learned world model.
 1414: For our dataset collection, we train a TD-MPC2 agent for each MetaWorld environment from scratch
 1415: with the default hyperparameter settings which can be found in Appendix H of the original paper.
 1416: We report the episode returns as a function of environment steps in Figure 11. We train each agent
 1417: for 1M environment steps as we find that to be sufficient for the agent to learn to solve the task
 1418: consistently. We store the low-dimensional state and RGB image observations from the replay
 1419: buffer, which consist trajectories of varying expertise from a random uninitialized policy to a fully
 1420: trained performant policy. Our final offline dataset consists of 1000 trajectories of 100 timesteps
 1421: across four different MetaWorld tasks.
